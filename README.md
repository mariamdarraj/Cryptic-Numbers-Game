# About
Cryptic Numbers is a 2D game where the player tries to guess a random number within a range.  
The game tracks the number of attempts and wins, and allows the player to select difficulty levels: Easy, Medium, or Hard.

# Features
- Choose difficulty: Easy (1-100), Medium (1-500), Hard (1-1000)
- Track total wins and attempts
- Interactive GUI with buttons and input fields
- Clean and rounded UI components
- Sound and vibration toggle (settings panel)

# Technologies Used
- Java
- Swing
- OOP (Object-Oriented Programming)

# How to Run
1. Clone the repository:
2. Open the project in your preferred Java IDE (IntelliJ, Eclipse, NetBeans)
3. Run `NumberGame.java`
